# @reket/ui

### REKET UI LIBRARY

## Info

UI Library Based on `React` and `Tailwind CSS`

## Install

```sh
npm install @reket/ui
```

or

```sh
yarn add @reket/ui
```

## Usage

```typescript
import { Button } from '@reket/ui';

const YourOwnComponent = () => {
    return (
        <Button text="Button Text" color="purple" size="xl" option="filled" />
    );
};
```
